## Overview
Optimizing training volume requires balancing the stimulus for muscle adaptation against the systemic fatigue generated by the workout. This project uses a modified **Banister Fitness-Fatigue model** as a simplified proxy to mathematically identify optimal training intervals and volumes. 

Rather than relying on static performance snapshots, this simulation executes a multi-variable grid search to maximize a custom **Optimization Score** (the 14-day trailing performance average minus a max-fatigue penalty) while keeping weekly volume constant.

## Features
* **Exponential Decay Modeling:** Calculates dynamic fitness and fatigue curves using isolated $k$ coefficients.
* **Grid Search Optimization:** Iterates over multiple weekly volumes (100, 150, 200) and training intervals (1 to 7 days).
* **Algorithmic Efficiency:** Utilizes $\mathcal{O}(N)$ time-complexity `numpy` array pre-allocation.
* **Interactive Visualization:** Includes data matrices, `matplotlib` plotting, and dynamic `ipywidgets` to visualize the optimal frequency in real-time.
